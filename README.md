# Om Lathiya - Backend Engineer Portfolio

This repository contains the source code for my personal backend engineering portfolio, built as a single-page React application and deployed with GitHub Pages.

Live site: https://lathiyaom.github.io/Portfolio/

## Purpose

The portfolio is designed to:
- Explain backend value quickly
- Focus on business outcomes instead of buzzwords
- Show engineering depth without overwhelming non-technical readers
- Build trust with practical, measurable proof

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React
- Lenis (smooth scrolling)

## Local Development

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

Build production bundle:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
